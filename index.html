About this data

The City of Melbourne Pedestrian Counting System has recorded foot traffic across the CBD since 2009. Sensors are installed under awnings and on street poles, recording bi-directional pedestrian movements 24 hours a day. No images or personal information are collected — only movement counts.

Sensor locations are selected based on retail and event activity, regular pedestrian use, and entry/exit flows to key areas of the city.

Data sources

Historical average — Pedestrian Counting System, monthly hourly counts. Used to calculate % above/below typical traffic for this time of day.
Sensor locations — Pedestrian Counting System Sensor Locations. 137 active sensors across the City of Melbourne municipality.
How to read the map

Each sensor appears as a hexagon. Colour and size indicate pedestrian intensity:

Tap any hexagon to see the exact count and comparison to the recent average. Use the Rankings tab to see every sensor ordered from busiest to quietest.

How data is refreshed

A Python script runs on a local Windows machine via Windows Task Scheduler. It fetches current data from the City of Melbourne Open Data API, builds a static JSON file, and commits it to GitHub. This page loads that file directly — there are no live API calls from the browser.
