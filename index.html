Data Sources & Method
Live counts — Pedestrian Counting System, past hour (per minute). data.melbourne.vic.gov.au
Averages — Same hour, same day-of-week, over the last 4 weeks. Gives a like-for-like "typical Sunday at 11am" baseline.
Hourly profile & peak hours — Full day of hourly data from yesterday. 137 active sensors across the City of Melbourne.
Refresh method — Python script on Windows Task Scheduler pushes a static JSON to GitHub. No live API calls from the browser.
